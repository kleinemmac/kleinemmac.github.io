# Primary Portfolio
This is my second (and final) attempt at a portfolio page. I wanted something much more minimalist and much less 90's teenage blogger.

## Purpose:
The purpose of this project was to create a professional, modern portfolio site.

## Technologies
HTML, CSS, and Adobe xD

Upon finalizing design compositions for this site, I realized that there was no exigence for libraries to aid in layout and design. I wanted a simple and clean approach for a simple and clean site.

## Things I Learned

- you can make a lot with a little (don't use libraries unless they are necessary)
